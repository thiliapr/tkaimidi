# Tk AI MIDI
## 简介
这是一个MIDI生成模型，旨在通过深度学习技术生成音乐作品。

## 安装与依赖
```bash
pip install -r requirements.txt
```

## 训练
```bash
python3 tokenizer.py ${samples_path}  # 训练分词器
python3 train.py ${num_epochs} ${train_dataset_path}  # 训练模型
```
